# Planification Chasse au Trésor

Date : 2025-10-31

## Structure de la Chasse

Chaque indice mène à une cachette où se trouve l'indice suivant.

## Cachettes Disponibles dans la Maison

Voici des idées de cachettes courantes :

1. **Cuisine**
   - Dans le four
   - Dans le frigo (tiroir à légumes)
   - Dans une boîte de céréales
   - Sous un torchon
   - Dans le micro-ondes

2. **Salon**
   - Derrière un coussin du canapé
   - Dans un tiroir de la télé
   - Sous un tapis
   - Dans un livre spécifique
   - Derrière un cadre photo

3. **Salle de bain**
   - Dans l'armoire à pharmacie
   - Sous une serviette
   - Dans le panier à linge
   - Derrière le miroir

4. **Chambres**
   - Sous l'oreiller
   - Dans une chaussure
   - Dans l'armoire à vêtements
   - Sous le lit
   - Dans un tiroir de bureau

5. **Entrée/Couloir**
   - Dans un manteau/veste
   - Dans le placard à chaussures
   - Derrière la porte

6. **Autres**
   - Dans la boîte aux lettres
   - Sur le balcon/terrasse
   - Dans le garage
   - Dans la machine à laver

## Format de Chaque Indice

```
╔════════════════════════════════════════════╗
║           INDICE N° [X]                    ║
║                                            ║
║  [TEXTE DE L'INDICE - 1 à 2 lignes]       ║
║                                            ║
║  Cachette de cet indice : [LIEU]          ║
╚════════════════════════════════════════════╝
```

## Conseils pour les Indices

- **Énigmes simples** : Rimes, devinettes, descriptions
- **Références personnelles** : Objets familiers de la maison
- **Progresser logiquement** : Éviter les allers-retours excessifs
- **Adapter la difficulté** : Selon l'âge du participant

## Exemple de Séquence (5 étapes)

1. **Indice 1** (point de départ) → mène au **four**
2. **Indice 2** (caché dans le four) → mène au **canapé**
3. **Indice 3** (caché sous coussin) → mène à **l'armoire à vêtements**
4. **Indice 4** (caché dans armoire) → mène au **frigo**
5. **Indice 5** (caché dans frigo) → mène au **TRÉSOR FINAL**
